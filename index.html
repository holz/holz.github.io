使用 Hexo 在 GitHub 搭建博客

关于 github

关于账号申请和设置

申请 GitHub 账号，设置 ssh key，下载安装 git bash、 node.js 之类网上已有大把资料，也不容易出错。

关于建立仓库

登陆 github 之后，因为要建立的是一个博客，所以在 New repository 那里，取名要以 xxx.github.io 的形式。

譬如，我希望别人用 holz.github.io 这个网址访问我的博客，那么在 New repository 的时候，取名就是 holz.github.io。

之前就是没理解，取名错误，走了弯路。

关于 hexo

之前跟着一个很详细的教程做，结果一步一步下来，居然没办法部署，这一定是我看的姿势不对。

所以到 hexo 官网 看，然后结合自己的理解，重新做，总算把网站搞成了。

建立 hexo 博客本地版
